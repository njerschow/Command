Features

Built for developers who live in the terminal

Smart summaries

AI-generated 5-word descriptions via Claude. Local heuristics handle common patterns first.

Instant switching

Click any row or press ⌘+1–9 to jump straight to the right window and tab.

Action required

Pulsing indicator when a terminal needs attention — prompts, errors, or Claude waiting for input.

Session history

Closed terminals are saved with summary and output. Search and restore past sessions anytime.

Claude Code aware

Tracks Claude session state via hooks. See working, waiting, or needs-permission at a glance.

Battery friendly

Adaptive polling, fingerprint-based change detection, batched AI calls. Nearly zero CPU usage.

How it works

Three layers, zero config

1

Scan

Every 2s, reads Terminal.app and iTerm2 windows via AppleScript. Detects running processes.

2

Fingerprint

Periodically reads terminal output, strips ANSI noise, and hashes for change detection.

3

Summarize

Local heuristics handle SSH, builds, and servers. Ambiguous terminals go to Claude in one batch.
