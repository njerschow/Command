How it works

1

Scans every 2 seconds

Detects all Terminal.app windows and tabs via AppleScript. Reads process state and window titles.

2

Fingerprints every 5 minutes

Reads the last 100 lines, strips noise (spinners, progress bars, ANSI codes), and hashes for change detection.

3

Summarizes intelligently

Local heuristics handle obvious cases (SSH, servers, builds). Only ambiguous terminals get sent to AI — one batched call for all of them.
